Configuración
Conexión Google Sheets

Pega la URL del Web App de Google Apps Script para activar la sincronización en tiempo real con tu planilla compartida.

Cómo configurar (5 pasos)
  1. Crea un Google Sheets nuevo y nómbralo "Finanzas Poritos"
  2. Ve a Extensiones → Apps Script
  3. Borra el código que aparece y pega el contenido del archivo Code.gs
  4. Ejecuta la función inicializarHojas() una vez (botón ▶ Run)
  5. Ve a Implementar → Nueva implementación → Aplicación web
  6. En "Quién tiene acceso" elige "Cualquier persona", copia la URL y pégala arriba
  7. Comparte el Google Sheet con tu pareja con permiso de edición
